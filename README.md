## Overview
This project is an Excel-based client payment tracking system designed for personal trainers managing both online and in-person clients. It supports subscription-based payments, session packages, and pay-as-you-go transactions, with an automated dashboard for income analysis.

## Problem Statement
Personal trainers often manage multiple client types and payment structures, making it difficult to track income clearly. This project provides a simple but structured solution using Excel.

## Features
- Client payment entry sheet
- Support for online and in-person clients
- Monthly income calculation
- Automatic income breakdown by client type
- Interactive dashboard with charts
- Error handling for empty or missing data

## Tools Used
- Microsoft Excel (Web / Desktop)

## Dashboard Insights
- Total number of unique clients
- Total income
- Monthly income trends
- Online vs In-person income comparison

## Intended Users
- Personal trainers
- Coaches
- Small fitness businesses
- Excel learners practicing real-world analysis

## How to Use
1. Enter client payments in the `Client_Payments` sheet
2. Select client type and payment type
3. Dashboard updates automatically
